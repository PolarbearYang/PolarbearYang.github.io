该模块包含的内容主要有：

  • 包括图像修复、扣图、超分辨率和四种图像生成四种类型的任务
  • 可以对上述四种任务类型进行单图推断产生demo并可视化

命令行调用部分

训练

dist_train.sh(multi-gpu training)

该文件位于/mmlab/mmediting/tools/dist_train.sh，其参数如下：

必要参数
  • WORK_DIR 参数为任务路径。
主要可选参数
  • GPUS 使用的GPU数量，默认值为1。
  • CONFIG 使用的config文件，默认使用工作路径下的config.py文件。
  • PORT （可选）使用的端口编号。格式为PORT:-29500。
次要可选参数
  • --resume-from （可选）如果有之前中断的训练xxx.pth文件，参数为该xxx.pth文件的路径。从该模型继续训练。
  • --no-validate （可选）无参数。若有此选项，则训练时不显示验证集结果。
  • --deterministic （可选）无参数。CUDNN中backend中的某个选项。
  • --launcher （可选）参数为none, pytorch, slurm, mpi中的一个。选择job launcher的类型。
  • --autoscale-lr （可选）无参数。决定学习率是否根据使用的gpu数量自动调整。
测试

dist_test.sh(multi-gpu testing)

该文件位于/mmlab/mmdetection/tools/dist_test.sh，其参数如下：

必要参数
  • WORK_DIR 参数为任务路径。
主要可选参数
  • GPUS 使用的GPU数量，默认值为1。
  • CONFIG 使用的config文件，默认使用工作路径下的config.py文件。
  • PORT （可选）使用的端口编号。格式为PORT:-29500。
次要可选参数
  • --seed （可选）选择的随机数种子。
  • --deterministic （可选）无参数。决定是否使用GPU收集结果。
  • --gpu_collect （可选）无参数。决定是否使用GPU收集结果。
  • --save-path （可选）存储结果图片的路径
  • --launcher （可选）参数为none, pytorch, slurm, mpi中的一个。选择job launcher的类型。
  • --local_rank （可选）参数是一个int。似乎与分布式训练相关。
